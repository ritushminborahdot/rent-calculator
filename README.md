🧮 Rent Split Calculator (Python)
This is a simple Python project that helps people calculate and split living expenses equally among roommates or hostel mates.

The program asks users to enter different expenses like house rent, food bill, electricity bill, and other expenses, and then divides the total amount by the number of people living together.

• It is useful for students, roommates, or anyone sharing expenses.

🚀 Features:
• Enter house/hostel rent

• Enter total food bill amount

• Enter electricity bill

• Enter other expenses

• Enter number of persons

• Automatically calculates the total expense

• Splits the amount equally among all persons

🛠️ Built With
Python (Basic Python concepts)
